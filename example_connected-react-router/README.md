# React Boilerplate CRA Template Examples & Tutorials

## Connected React Router

[`connected-react-router`](https://github.com/supasate/connected-react-router) is not provided out of the box. The reason is simply because the `react-router` [doesn't suggest it](https://reacttraining.com/react-router/web/guides/redux-integration/deep-integration). For those who want to use it here is the example setup.
